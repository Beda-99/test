What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create webpages. HTML structures content like text, images, links, videos, and forms.

HTML is not a programming language — it is a markup language that tells the browser how to display content.

Why Learn HTML?

✔ It is the foundation of all websites.

✔ Easy to learn for beginners.

✔ Required for web development.

✔ Works with CSS and JavaScript.

HTML Attributes

What Are Attributes?

Attributes give extra information about HTML elements. They are placed inside the opening tag.

<a href="https://example.com">Visit</a> <img src="image.jpg" alt="My Image">

Common Attributes

href – Link destination

src – Image source

alt – Alternative text

class – Used for styling

id – Unique identifier

Advanced HTML Forms

Form Input Types

Form Validation
